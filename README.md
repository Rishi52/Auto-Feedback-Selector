# Auto Feedback Selector Chrome Extension

A simple Chrome extension that automatically selects feedback options like:

- Neutral
- Agree
- Strongly Agree

## 🚀 Features
- One-click auto selection
- Clean and minimal UI
- Works on most feedback forms using radio buttons

## 📦 Installation

1. Clone this repository
2. Go to `chrome://extensions/`
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select the project folder

## 🛠 Built With
- HTML
- CSS
- JavaScript
- Chrome Extension Manifest v3
